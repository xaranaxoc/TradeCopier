# UI Redesign Pilot — CustomTkinter

Этот PR добавляет **автономный preview** нового дизайна GUI (`gui_ctk_pilot.py`),
параллельно со существующим `gui.py`. Текущий продакшен-GUI **не затронут**.

## Запуск

```bash
pip install customtkinter
python gui_ctk_pilot.py
```

Используются dummy-данные — никаких подключений к MT5, лицензии или копитрейдеру.

## Что в пилоте

- **HeaderBar** — логотип, заголовок, два блока действий
  (КОПИТРЕЙДЕР / ТЕРМИНАЛЫ) и icon-кнопки настроек/инфо.
- **MasterPanel** — путь к терминалу, action-иконки, метрики
  (Логин / Баланс / Эквити / P&L) в одной скруглённой карточке.
- **KPI-карточки** — Master Balance, Total Equity, Net P&L, Сделок скопировано.
- **SlaveTable** — пять примеров аккаунтов в `CTkScrollableFrame`,
  каждая строка скруглена, цветная боковая полоса по статусу.
- **BottomTabs** — `CTkTabview` с тремя вкладками: Сделки / Логи / Статистика.
- **SlaveDialogPreview** — пример скруглённого модального диалога
  с полями и слайдером риска (вызывается из «+ Добавить slave» и «⚙»).

## Палитра

Сохранена фирменная neon-cyan палитра из существующего `gui.py`
(`ACCENT=#00B4D8`, тёмные фоны `#080810 / #0C0C14 / #11111C`).
Добавлены сглаженные углы (`corner_radius` 10–14 px), более воздушные
отступы и более крупная типографика для значений.

## Что отличается от текущего UI

- **Углы** — CTk рисует настоящие rounded corners (без эмуляции через Canvas).
- **Кнопки/инпуты** — toggle-свитч вместо ☑ символа, скруглённые pill-кнопки.
- **Цветовая иерархия** — единые `PillButton`/`IconButton`/`KpiCard`,
  вместо россыпи `tk.Button(...)` с разными параметрами.
- **Таблица** — на `CTkScrollableFrame`, каждая строка — отдельная
  скруглённая карточка с боковой статус-полосой.

## Что НЕ сделано в этом PR (намеренно)

- Связка с реальным `CopyTrader` / `MT5` — пилот только визуальный.
- Перенос диалогов SymbolPicker / Settings / Activation — только sample.
- pystray / лицензия / обновлятор — не трогаем.
- Профили, paste, темы — будут в следующих PR.
